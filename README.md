Personal Expense Tracker

This is a simple Python script to help you track your daily expenses. It saves all the data into a text file so you can view your history later.

Features

Add Expense: Enter the date, category, amount, and a description.

View Expenses: Shows a list of everything you have spent so far.

Calculate Total: Adds up all the expenses to show the total money spent.

Data Persistence: Saves data to expenses.txt automatically.

How to Run

Make sure you have Python installed.

Download expense_tracker.py.

Open your terminal or command prompt.

Run the script:

python expense_tracker.py


How it Works

The program creates a file called expenses.txt in the same folder. It stores data in a format like Date,Category,Amount,Description.

Note: If you delete expenses.txt, you will lose your saved history!
